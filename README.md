Σκοπός της εργασίας αυτής είναι η υλοποίηση και μελέτη ενός συστήματος εντοπισμού θέσης για ένα τροχοφόρο ρομποτικό όχημα στο περιβάλλον προσομοίωσης CoppeliaSim (VREP) με χρήση της γλώσσας Python. Η εκτίμηση της θέσης επιτυγχάνεται συνδυάζοντας το μοντέλο κίνησης και τις μετρήσεις από ορόσημα (landmarks) μέσω του αλγορίθμου Extended Kalman Filter (EKF).
